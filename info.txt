
Funcionalidades:

Adicionar nomes: 
    
    Os usuários escreverão o nome de um amigo em um campo de texto e o adicionarão a uma lista 
    visível ao clicar em "Adicionar".
    Capturar o valor do campo de entrada - utilize document.getElementById ou document.querySelector para obter o texto inserido pelo usuário. FEITO

Validar entrada: FEITO
    
    Se o campo de texto estiver vazio, o programa exibirá um alerta solicitando um nome válido,
    um alerta com a mensagem de erro: "Por favor, insira um nome."
    Implemente uma validação para garantir que o campo não esteja vazio.
  

Visualizar a lista: 
    
    Os nomes inseridos aparecerão em uma lista abaixo do campo de entrada.
    Se o valor for válido, adicione-o ao array que armazena os nomes dos amigos usando o método .push().

Sorteio aleatório: 
    
    Ao clicar no botão "Sortear Amigo", um nome da lista será selecionado aleatoriamente e exibido na página.


Limpar o campo de entrada: 
    
    Após adicionar o nome, redefina o campo de texto para uma string vazia.
